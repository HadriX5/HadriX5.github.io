Hybrid AI Phishing Detector

View Repository →

Developed a cutting-edge security system that identifies malicious URLs by combining Reinforcement Learning (Q-Learning) with K-Nearest Neighbors (KNN). This hybrid architecture was designed to overcome the "state space explosion" in RL, achieving a remarkable 99.56% accuracy on the PhiUSIIL dataset by balancing deterministic memory and supervised generalization.

The core engineering challenge involved building a feature discretization pipeline using quantile-based binning to transform continuous URL and HTML data—such as character transition rates and domain-title similarity scores—into discrete states. The system operates with high efficiency: it uses the Q-table for constant-time O(1) lookups on known states and only triggers the KNN fallback layer when encountering unseen patterns, ensuring real-time performance without sacrificing precision.

UniReview Android App

View Repository →

Developed a comprehensive academic review platform for the Android ecosystem, designed to facilitate transparency in university course evaluations. The core of the application is built upon a strict 4-layer Clean Architecture (Presentation, Domain, Feature, and Data), ensuring that business logic remains independent of external frameworks or UI changes.

I implemented Reactive Programming using RxJava to manage complex asynchronous data streams, significantly reducing UI thread blocking during network operations. For data persistence and authentication, I integrated Firebase Services, while maintainability was enhanced through rigorous Unit Testing of use cases and domain entities.

MovieLens Recommendation Engine

View Repository →

Designed and implemented a high-performance Collaborative Filtering engine using the MovieLens-1M dataset. The project focused on optimizing computational efficiency by leveraging NumPy vectorized operations, replacing traditional iterative loops with matrix-based similarity calculations (Euclidean and Cosine distance).

Beyond the algorithm, the project involved an in-depth demographic analysis to detect rating bias. I engineered a data pipeline to identify behavioral discrepancies across gender and age groups, visualizing these patterns using Matplotlib and Seaborn. The model's accuracy was rigorously validated using Mean Absolute Error (MAE) metrics across different sparsity levels.

Open n-gram Language Detector

View Repository →

Built a robust Language Identification (LI) system from the ground up, implementing a custom Multinomial Naive Bayes classifier without relying on high-level ML libraries for the core logic. The engine is designed to handle high-dimensional sparse data by calculating log-probabilities and applying Laplace Smoothing to prevent zero-probability issues with unseen tokens.

A key innovation was the implementation of Open n-grams (skip-grams) for feature extraction, which allows the model to capture morphological patterns even in the presence of noise or typos. This approach significantly improved performance on short-text classification compared to standard character n-grams. The system includes an explainability layer to visualize the weight of specific tokens in the final prediction.
